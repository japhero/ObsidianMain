The rule only applies if $\displaystyle \lim_{ x \to c } f(x) = \lim_{ x \to c } g(x) = 0$. This means that we can only apply this rule when we get a limit of $\frac{0}{0}$.
# Formula

$$\lim_{ x \to c } \frac{f(x)}{g(x)} = \lim_{ x \to c } \frac{f'(x)}{g'(x)} $$
