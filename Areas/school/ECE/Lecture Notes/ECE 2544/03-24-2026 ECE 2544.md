## Register Transfers
- As the system grows it becomes impractical to design with state tables 

- Break it down into the 
	- *datapath*: performs data processing operations
	- *control unit*: determines the sequency of the operations

- Register transfers represent the movement and processing of data. Each register transfer operation has three basic operations
	- a set of registers
	- The set of micro operations: operations that can be performed on a single register
	- The control functions: govern when the micro operations are performed

### Applications of Logic Microoperations


**Selective Complement**
- put 1's where we want to force 1's 
- put 0 where we want to keep the value 
- OR every bit with the complement


**BIt Mask**
- put 0's where we want to force the value to zero 
- put 1's where we want to keep the value 
- AND every bit with the mask 



#### Shift micro operations 
- Logical shift fills the vacant position with 0
- a circular shift fills the vacant positions with wrapped values  
	- CSR: $0101 \to 1010$ 


### Bus system source design

## Floating Point numbers

i has 3 components
- $r$ radix
- m the mantissa 
- e the exponent

- IEEE 754 is a standard for floating point representation 
	- 32 bits (single precision) 
		- 1 sign bit 23bit mantissa 8 bit exponent 

$$
\mathrm{Value}=(-1)^s \times 1.F \times 2^e
$$
- F is the fractional part of the mantissa 

**Exponent**
- The exponent gets stored using the excess 127 format 
	- This means that we just add 127 to each number representation to create a larger range 

**Mantissa (fractional)**
- the mantissa only stores the fractional part of the number


**Steps**
- Convert the number to binary
- Normalize the number by moving the decimal to the right of the MSB
- Determine S,E,F

> [!Example]- $+5.25$ to float
> 
> $$
> \begin{align}
>  & 5.25 \\
> \mathrm{binary: \space} & 101.01 \\
> \mathrm{\#\mathrm{Normalized: \space}} &1.0101 \\
>   S= &  0 \\
>   m= &  0101\dots 0 \\
>   e=  & 1000 \space 0001    \\
>   \mathrm{HEX} =  & 40A{8}0000
> \end{align}
> $$
