## K maps 

- A Karnaugh map is a diagram made up of squares
	- Each square represents one possible minterm of a function.
	-  The squares are labeled with their minterm number. For example, ABC = 101 is minterm 5.
	- We enter a 1 or a 0 into each box of the Karnaugh map based on the function’s output for that literal combination.
	
	
- Like gray codes only one bit can change when going either vertically or horizontally when going along the kmap

![center|300](https://media.geeksforgeeks.org/wp-content/uploads/20241206120531647347/_variable-K-map-for-SOP-and-POS.webp)


### Minimizing with Kmaps 

- We group together rectangles/squares that contain 1 and are adjacent 
	- The number of cells in each group must be a <u>power of 2 </u>
	- Maximize the size of groups 
	- minimize the number of groups 
	- Each group must have one cell that isnt in any other group
