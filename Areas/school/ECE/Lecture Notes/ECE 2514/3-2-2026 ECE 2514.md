### Structs
- A struct is a custom data type 
- A struct has a type declaration
- A struct is a `class type`
	- i dont think this means it is a class rather it is a child of the basic class type instance 
- 

**Struct Example**
``` cpp
struct Employee
{
    int id {};
    int age {};
    double wage {};
};
```
- variable that are part of the struct are called members 
- Within the example the curly braces ensure that the members are initialized 
- To and modify members we use the member selection operator `.` (period). Like methods from a class. 

### Struct aggregation
- an aggregate data type (called aggregate) is any type that can contain multiple data types.

**Aggregate Initialization**
``` cpp
struct Employee
{
    int id {};
    int age {};
    double wage {};
};

int main()
{
    Employee frank = { 1, 32, 60000.0 }; // copy-list initialization using braced list
    Employee joe { 2, 28, 45000.0 };     // list initialization using braced list (preferred)

    return 0;
}
```
